Beberapa Shortcut Linux dan Vim

  • Min 07 Juli 2019
  • Programming
  • Zarkasy

Bagi yang programmer pasti lama kelamaan coba-coba pakai linux dan vim. Memang ga tau kenapa untuk programmer ini linux ini merupakan OS yang sering dipakai, mungkin karena beberapa source code hanya bisa dicomplie di linux, ini sih sebenarnya alasan mengapa saya beralih ke linux walaupun masih juga pakai windows. Nah disini saya akan berbagi beberapa shortcut yang berguna banget untuk produktifitas dalam programming.

Linux shortcuts

  1. alt-tab : Untuk pindah antar aplikasi. Berguna banget misal pas coding kita butuh untuk browsing sesuatu, daripada pakai mouse mending pakai shorcut ini.
  2. windows + arrow : Untuk memposisikan window aplikasi kita secara split. Untuk posisi splitnya tergantung arrow yang dipencet, bisa posisi kanan, kiri, atas atau bawah.

Vim shortcuts

  1. gt : Pindah next tab. Shotcut ini bener-bener baru tau, karena sebelumnya pakai vim di windows biasanya ga pernah pakai tab, biasanya pakai buffer aja.
  2. tabnew : buka tab baru

Mungkin itu aja, semoga bermanfaat.
