Pemahaman Awal Tentang Janus

  • Sen 08 Juli 2019
  • Janus, Programming
  • Zarkasy

Janus adalah webrtc server, aplikasi yang bertugas dalam mengatur koneksi webrtc antar client. Namun kekurangan software ini adalah dokumentasinya yang kurang jelas. Disini saya akan menuliskan pemahaman awal bagaimana setup janus.

1. Server janus dinyalakan.
2. Setup client janus dengan janus.js. Ketika success kita attach pluginnnya.
3. Jika success attach plugin, baru kita memiliki handle yang bisa untuk mengotak-atik plugin api.
4. Untuk melakukan operasi plugin, janus menggunakan json dan fungsi handle.send({//json}). Disini kita bisa melakukan operasi list group, create group, delete group, dsb.
