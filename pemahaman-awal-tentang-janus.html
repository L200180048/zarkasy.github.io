Pemahaman Awal Tentang Janus

  • Sen 08 Juli 2019
  • Janus, Programming
  • Zarkasy

Janus adalah webrtc server, aplikasi yang bertugas dalam mengatur koneksi webrtc antar client. Janus adalah open source, dan dikembangkan oleh beberapa orang saja. Jika saya lihat, karena SDMnya terbatas maka walaupun software ini bagus namun kurang dalam hal dokumentasi. Kualitas dokumentasi yang dibuat terdapat pada level medium kebawah, maksud saya kamu bisa mengerti bagaimana menggunakan janus jika kamu membaca dokumentasi dan membaca source codenya. Nah disini saya kan menuliskan pemahaman awal saya tentang bagaimana menggunakan janus.

  1. Jalankan server janus. Untuk menjalankan server janus kamu membutuhkan OS linux/Mac dan mengintsall janus sesuai dengan petunjuk install. Software ini tidak bisa diinstall menggunakan satu klik saja atau satu command saja namun harus mengikuti petunjuk diatas.
  2. Buat file html dengan mengattach file adapter.js, janus.js. Untuk template dari sisi client bisa dilihat disini.
  3. Init janus beserta konfigurasinya, jika suksek terhubung dengan janus server, maka kita bisa mengattach plugin yang mau kita pakai.
// Init janus
var pluginhandle = null;
var janus = new Janus({//conf},
    success: function(){
        // Attach plugin, bisa videoroom, audiobridge dll
        janus.attach({
            plugin:"janus.conf.videoroom"
        },
        success: function(handle){
            pluginhandle = handle;
        }
        );
    },
    onmessage:function(event){

    }
);
  1. Setelah sukses mendapat handleplugin. Disini kita bisa mulai melakukan request misal untuk membuat room untuk videoroom/videoconference. Jika room sudah sukses dibuat, maka kita join, setelah sukses join baru kita mempublish media(video/audio) diroom tsb.
pluginhandle.send(); //request create room
