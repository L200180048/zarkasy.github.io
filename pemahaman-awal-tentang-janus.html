Janus adalah webrtc server, aplikasi yang bertugas dalam mengatur koneksi webrtc antar client. Namun kekurangan software ini adalah dokumentasinya yang kurang jelas. Disini saya akan menuliskan pemahaman awal bagaimana setup janus. 1. Server janus dinyalakan. 2. Setup client janus dengan janus.js. Ketika success kita attach pluginnnya. 3. Jika success attach plugin, baru kita memiliki handle yang bisa untuk mengotak-atik plugin api. 4. Dari handleplugin ini misal pada videoroom kita bisa request untuk create room, dsb.
