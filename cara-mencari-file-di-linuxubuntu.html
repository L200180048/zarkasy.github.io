Cara Mencari File di Linux/Ubuntu

  • Kam 18 Juli 2019
  • Linux
  • Zarkasy

Buat kamu yang memakai linux dan sering memakai command line. Ada satu fungsi pencarian yang menurutku sangat powerful untuk digunakan yaitu pipeline antara find dan grep. Kita tau kalau dilinux terdapat banyak sekali command, dan cukup merepotkan kalau kita harus mengingat banyak sekali command tersebut, tak terlepas command find itu sendiri, command ini memiliki banyak options yang bisa digunakan dibelakangnya, misalkan kita mau mencari suatu file dengan nama tertentu dan dengan size lebih dari sekian. Namun sering kali kita hanya ingin mencari file dengan menginputkan nama, untuk itu kita bisa menggunakan gabungan fungsi find dan grep. Caranya adalah sebagai berikut

find [path] | grep [nama file] // find /home/user/ | grep "fileku"

Penjelasan dari command diatas adalah, command find [path] akan menghasilkan daftar file yang terdapat dalam direktori tersebut, dan grep [nama file] memfilter hasil dari find. Demikian cara untuk mencari file dalam linux, semoga bermanfaat.
