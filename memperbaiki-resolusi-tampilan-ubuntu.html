Memperbaiki Resolusi Tampilan Ubuntu

  • Tue 09 July 2019
  • misc

Ubuntu linux menurut saya memiliki kelebihan dibandingkan windows, namun tidak menutupi bahwa windows juga memiliki kelebihan dibandingkan linux. Salah satu kelebihan windows yang saya lihat adalah kemudahan dalam menginstall driver, berbeda dengan ubuntu terkadang menginstall driverpun sudah rumit dan setelah driver diinstallpun masih ada masalah. Kasus yang saya alami adalah tampilan monitor ubuntu terlalu rendah padahal driver sudah diinstall. Solusi yang saya gunakan adalah dengan menggunakan xrandr. Caranya bisa dilihat pada link ini
